Paired Stimulus Preference Assessment Tool

Welcome to the Paired Stimulus Preference Assessment Tool. This program follows the paired stimulus preference assessment methodology and was designed to allow users to systematically identify stimulus preferences. The tool allows users to assess 3–10 videos per assessment. The users can customize each assessment by adjusting the following parameters: time to stimulus selection, stimulus location, stimulus presentation of samples (still images or videos), and duration of playtime for selected stimuli. The tool will calculate and display data regarding the percentage of selection per video, percentage of selections per location (left/right or top/bottom), and assessment duration at the conclusion of each assessment.

Instructions

  1. Upload stimuli. This parameter allows users to upload YouTube videos. To add a video, copy the YouTube URL and paste it in the URL textbox. Click inside the title textbook and add a name. Click the Add Stimulus button. The video title will appear in the Stimulus Library.
  2. Select videos for assessment. This parameter allows user to select the stimuli for an assessment. To select a stimulus, click the box next to each video title in the Stimulus Library. The program allows users to select 3–10 stimuli per assessment. The Stimulus Library will store up to 20 stimuli.
  3. Time to selection. This parameter allows users to select the duration of time allotted for a response. The duration starts each time a stimulus is presented on the screen. The value can range from 3–90 seconds.
  4. Location of stimuli. This parameter allows users to arrange stimuli horizontally or vertically.
  5. Stimulus option displays. This parameter allows users to display the stimulus options as still images or as videos without sound.
  6. Playtime for selections. This parameter allows users to select the duration of time selected stimuli will play after each selection. The value ranges from 0 to 90 seconds.
  7. Start assessment. Click the Start Test button.
